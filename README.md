# PhaseOpenGL

# Визуализация фазовых портретов на OpenGL

Интерактивная программа для построения фазовых портретов двумерных автономных систем дифференциальных уравнений.

## Возможности

- Встроенные системы: линейная, Ван-дер-Поль, Лотка–Вольтерра.
- Пользовательский ввод уравнений с параметрами (a–q) и функциями (sin, cos, exp, log и др.).
- Поле направлений с цветовой индикацией скорости.
- Численное интегрирование методом Рунге–Кутты 4-го порядка (вперёд/назад).
- Автоматический поиск и классификация точек равновесия (узел, фокус, седло, центр).
- Интерактивность: клик – добавление траектории, WASD – панорамирование, колесо – масштаб.

## Требования

- C++ 17, OpenGL 3.3+
- Библиотеки: GLFW, GLAD, GLM, Dear ImGui, tinyexpr
